DATASET SOURCE
==============

Name:      MovieLens 25M Dataset
Provider:  GroupLens Research, University of Minnesota
Released:  December 2019
Download:  https://grouplens.org/datasets/movielens/25m/
Direct:    https://files.grouplens.org/datasets/movielens/ml-25m.zip

The dataset files are not included in this submission. As stated in the
project brief, no real data collection is required and a sample dataset
from a public Big Data repository is sufficient. The dataset is public and
can be downloaded from the links above.

FILES USED
==========

ratings.csv   25,000,095 ratings   (userId, movieId, rating, timestamp)
movies.csv    62,423 movies        (movieId, title, genres)

Other files in the ml-25m package (tags.csv, links.csv, genome-scores.csv,
genome-tags.csv) were not used in this project.
